# Electron Typescript

A simple of electron app using typescript (frontend and backend)

## Production

Build application :

```bash
npm run build
```

Start application :

```bash
npm run start
```

## Development

Start frontend compilation watch :

```bash
npm run start:dev:frontend
```

Start backend compilation watch :

```bash
npm run start:dev:backend
```

Start application :

```bash
npm run start
```
